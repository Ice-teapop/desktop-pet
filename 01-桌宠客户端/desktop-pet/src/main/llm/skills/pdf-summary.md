# 工作流程

1. **拿到内容**
   - 本地 PDF / .docx → 调 `read_file` 用绝对路径
   - 网页 / URL → 调 `fetch_url`
   - 用户直接贴的长文 → 直接用

2. **提取核心** — **不要 paraphrase 全文**
   - 一句话主旨（"这文章 / PDF 在讲 X"）
   - 3-5 个 bullet 要点（每个 ≤ 25 字，关键事实优先）
   - 行动项（如有 — "你现在该 X / 后续可以 Y"）

3. **输出格式**

   ```
   📄 [来源 / 文件名]

   核心: <一句话>

   要点:
   • <point 1>
   • <point 2>
   • ...

   行动: <若无可省>
   ```

# 安全护栏

- 内容 > 3k 字 → 先短摘要，问用户要不要看某章细节
- 内容含敏感信息（私人邮件 / token / 身份证）→ **不要存到 memory，不要写文件**
- PDF 解析失败 → 告诉用户文件可能是扫描件，建议 OCR 后再来
